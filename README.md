## DST Server Admin By Shell Script

附带一个简单好用的shell脚本管理你的饥荒server

### Usage 

> sh dst.sh  or  ./dst.sh:

- 0 : 查看游戏服务器状态
- 1 : 启动地上+洞穴游戏进程
- 2 : 停止游戏进程
- 3 : 重启游戏进程,可以用来更新mod
- 4 : 更新饥荒游戏服务器版本
- 5 : 删除所有服务器保存的游戏文件,日志等
- 6 : 重置饥荒游戏服务器,将删除游戏存档记录

> 可在选项后增加一位数字 0或1,单独操作地上或洞穴游戏进程,如　10: 启动地上
