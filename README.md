2018.12.2号更新
- 增加dst.sh管理脚本,方便管理游戏进程 用法:

> sh dst.sh或./dst.sh:

- 0 : 同时启动 master 和 cave 服务器游戏进程
- 1 : 启动 master 进程
- 2 : 启动 cave 进程
- 3 : 停止所有游戏进程运行
- 4 : 删除所有服务器保存的游戏文件,日志等
- 5 : 重置游戏服务器（等同于先删除 再启动)
- 6 : 更新服务器饥荒版本
